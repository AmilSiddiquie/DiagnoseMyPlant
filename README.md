# DiagnoseMyPlant 🌿
An AI-powered Leaf Disease Detection System built with CNN and Flask.

## Group Members
* Amil Siddiqui
* Shahid
* Ehtesham
* Uzair

## Project Overview
This project uses a Convolutional Neural Network (CNN) to identify 39 different plant leaf diseases with 97% accuracy. It features a hybrid advice system that provides immediate treatment plans and includes a Gemini AI-powered chatbot for custom farmer queries.

## Technical Stack
* **Model:** CNN (TensorFlow/Keras)[cite: 1]
* **Backend:** Flask[cite: 1]
* **Deployment:** Docker & Hugging Face Spaces[cite: 1]
* **AI Chatbot:** Google Gemini API
